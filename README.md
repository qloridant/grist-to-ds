# Grist To DS

Permet d'assigner des labels à des dossiers de Démarche Simplifiée depuis Grist

# .env : exemples de variables attendues dans fichier .env :

LABEL_TABLE=LABELS _la table grist qui contient les labels à afficher ainsi que leur id DS_
DOSSIERS_TABLE=Demarche_128651_dossiers _la table des dossiers synchronisee par OTP_
DS_TOKEN = _votre token DS_
DS_TARGET=https://www.demarches-simplifiees.fr/api/v2/graphql _base path de l'api cible_
